# Third-party binaries (not distributed)

Place here before building the plugin:

- `ScriptHookRDRNetAPI.dll` — from the
  [ScriptHookRDR2DotNet-V2 release zip](https://github.com/Halen84/ScriptHookRDR2DotNet-V2/releases).

Runtime prerequisites in the RDR2 install folder (not needed for building):

- ScriptHookRDR2 (`ScriptHookRDR2.dll`) by Alexander Blade
- `ScriptHookRDRDotNet.asi` + `ScriptHookRDRNetAPI.dll` + ini from the same release zip

Built plugin DLLs go to `<RDR2>/scripts/`.
